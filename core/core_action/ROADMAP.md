# core_action Roadmap

## Mission
Provide stable action identity and trigger mapping contracts shared across host runtimes.

## Immediate Steps
1. Keep registration and trigger resolution deterministic.
2. Preserve simple fixed-capacity behavior with explicit failure on overflow.
3. Document adapter boundary for platform key events.

## Future Steps
1. Add contextual action scopes (global vs pane-local).
2. Add optional action categories and discoverability metadata.
3. Add optional conflict query helpers for trigger collisions.
